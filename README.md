# Python Suite Designer

Suite modular avanzada para diseño de interfaces UI/UX y GUI en Python.

## Estructura de carpetas

- `core/` - Núcleo de la aplicación, lógica principal, helpers.
- `ui/` - Componentes de interfaz, layouts, estilos.
- `widgets/` - Widgets personalizados y variantes.
- `plugins/` - Sistema de plugins/extensiones.
- `themes/` - Temas y estilos globales.
- `examples/` - Ejemplos y proyectos de muestra.
- `assets/` - Imágenes, íconos, fuentes, sonidos.
- `docs/` - Documentación generada.
- `tests/` - Pruebas unitarias y de integración.

## Requisitos

- Python 3.10+
- PyQt6
- PyInstaller
- GitPython
- watchdog
- jinja2
- jsonschema

## Instalación

```bash
python -m venv venv
source venv/bin/activate  # o venv\Scripts\activate en Windows
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```
